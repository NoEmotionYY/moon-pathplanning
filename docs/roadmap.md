# 路线图

## 第一阶段：基础工程和搜索算法

- 建立 MoonBit 项目结构、执行日志、测试目录与许可证。
- 实现网格模型、图结构、启发函数、BFS、DFS、Dijkstra、A 星和双向 A 星。
- 提供统一 Planner 返回结构。

## 第二阶段：JSON、CLI、SVG、benchmark

- 固化示例地图 schema 与基础 JSON 序列化。
- 提供演示 CLI 和 SVG 路径导出。
- 记录 benchmark 场景并为热点数据结构建立性能基线。

## 第三阶段：LPA*、D* Lite、RRT 系列

- 引入 LPA* 与 D* Lite 处理地图增量变化。
- 评估 RRT、RRT-Connect 与 RRT* 在连续空间中的建模边界。
- 补充更丰富的障碍物、采样与可视化案例。

## 第四阶段：mooncakes.io 发布与生态适配

- 收敛公开 API 与版本策略。
- 补充文档示例、兼容性说明和发布校验。
- 适配 MoonBit 生态中的依赖管理与包发布流程。
